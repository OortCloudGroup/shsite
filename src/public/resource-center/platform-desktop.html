平台概述

OortCloud Desktop，面向信创的跨平台桌面应用底座

基于 Electron 与 Tauri 双引擎，OortCloud Desktop 提供统一的桌面端应用运行环境，支持 Windows、Linux，并覆盖统信 UOS、OpenHarmony/鸿蒙桌面、银河麒麟等国产信创桌面操作系统，实现一次开发、多端适配与统一分发。

跨平台与信创支持

Windows / Linux / 统信 / 鸿蒙 / 银河麒麟统一适配，一套代码覆盖主流桌面。

自动更新

OTA 增量更新机制，用户无感知完成版本升级。

原生能力

系统托盘、通知、文件系统访问，深度集成桌面系统。

多窗口

支持多窗口并发运行，满足复杂业务并行工作流需求。

分发流程

从适配到灰度更新，五步完成桌面端分发

OORT 平台将目标系统识别、安装包构建、信创签名适配、企业通道分发与 OTA 更新串联为标准流水线，让 Windows、Linux 与国产信创桌面获得一致的安装和升级体验。

系统适配

识别 Windows、Linux 与信创桌面目标环境

多格式构建

生成 EXE/MSI、DEB/RPM 与信创安装包

签名加固

代码签名、完整性校验与国产系统策略适配

企业分发

应用市场、企业商店与离线包多通道投放

灰度更新

OTA 增量升级、分批发布与快速回滚

流程优化：分发前完成目标系统与 CPU 架构识别，分发中按企业策略选择在线市场、内网镜像或离线安装包，分发后通过遥测、灰度、回滚保障统信、鸿蒙、银河麒麟等信创桌面的持续可用。

平台特性

深度集成桌面端原生能力

OortCloud Desktop 充分利用各操作系统提供的 API，提供流畅、安全的桌面端体验。

系统托盘

后台常驻运行，随时待命不占用桌面空间。

原生通知

系统级消息推送，确保重要通知不被遗漏。

文件系统访问

直接读写本地文件，支持拖拽上传与批量操作。

多窗口

多窗口并行工作，提升复杂任务处理效率。

Desktop 扩展

专门面向利旧网页的 AI 化扩展

OortCloud Desktop 特别分发 Desktop 扩展，把 AI 助手能力带进既有网页系统：无需重写老系统，即可让 AI 读取页面上下文、执行网页操作、串联桌面工作流。

Legacy Web AI

让存量网页系统拥有“会看、会点、会填、会交付”的 AI 助手

借鉴浏览器智能扩展的能力模式，Desktop 扩展可以在授权范围内理解当前网页内容，辅助导航、点击按钮、填写表单、抽取数据，并与 OortCloud Desktop 的本地能力协同，把网页研究、业务录入、报表整理等流程从“人工复制粘贴”升级为“人审 AI 执行”。

利旧网页 AI 化页面上下文理解点击与表单辅助后台任务定时流程人工确认
浏览器侧感知读取可信网页中的内容、表格、按钮与表单状态。
桌面侧协同连接 Desktop 应用、本地文件与企业应用市场能力。
成果侧交付生成摘要、报表、台账或业务记录，减少手工搬运。

在旧系统中直接工作

面向 OA、CRM、ERP、BI 看板等存量 Web 系统，AI 可读取页面数据、定位入口、辅助完成重复点击与表单录入。

人工审批关键动作

对删除、提交、采购、财务、客户数据等敏感动作保留确认机制，先展示计划，再由用户审核后执行。

跨标签与后台任务

支持多页面资料检索、竞品信息汇总、后台巡检、日报周报等流程，任务运行时用户可继续处理其他工作。

定时与闭环交付

把每日数据抓取、例行检查、销售记录补全、邮件清理等固定流程编排为定时任务，并输出可追踪成果。
