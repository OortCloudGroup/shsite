平台概述

AxI 桌面端，统一门户与协同工作底座

AxI 统一工作空间为 OORT.sh 提供企业级桌面端入口。通过客户端深度集成，实现统一消息中心、工作空间标签、企业 SSO 与跨应用数据流的无缝协同体验。

统一消息中心

跨应用通知聚合，一站式消息管理。

工作空间标签

多任务并行，标签页高效管理。

企业 SSO

单点登录通行，一次认证全局访问。

跨应用数据流

应用间数据共享，无缝流转协同。

分发流程

AxI 桌面端部署条件

以下是部署 OORT.sh AxI 桌面端所需的技术条件与环境要求。

客户端版本AxI Desktop Client 3.0+
渲染内核Chromium 120+
开发语言JavaScript / TypeScript
SDKAxI SDK 3.x
通信方式WebSocket 长连接 / REST API
身份认证SSO (SAML 2.0 / OAuth 2.0 / OIDC)
分发方式AxI 客户端推送 / 工作空间配置 / 应用市场
